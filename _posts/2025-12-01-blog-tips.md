## Command

### Git Upstream

```shell
git remote add upstream https://github.com/cotes2020/jekyll-theme-chirpy
git remote -v
git fetch upstream
git checkout master
git merge upstream/master
```

### Run Server

```shell
bundle exec jekyll serve
```
