### 1. 배열

- 배열의 초기화뿐만 아니라 언어의 특징이나 안정성 등에 따라 초기화 규칙이 존재한다 
  - C언어 배열의 초기화 참고 [링크](https://en.cppreference.com/w/c/language/array_initialization)
  - 배열의 인덱스는 포인터 산술 연산 공식에 따라 계산되어, 메모리 기준 주소로부터의 상대적 거리(오프셋)로 처리되므로 인덱스의 크기와 관계없이 접근 속도에 차이가 없다

- 정적배열

  ```c
  const int arrLen = 3;
  int array[3] = { 0 };
  ```

- 동적배열
  
  ```c
  int arrLen = 4;
  int* pArr = (int*)malloc(sizeof(int) * arrLen);
  // 동적 배열은 메모리 해제에 신경 쓰자
  ```

- 배열의 길이
  
  ```c
  int len = sizeof(arrDeep) / sizeof(arrDeep[0]);
  ```

### 2. 다차원배열(정적, 동적)

```c
// 정적
int arr2D[2][3] =
{
  { 11, 12, 13 },
  { 21, 22, 23 }
};
// 동적
int rows = 3, cols = 4;
int **arr = malloc(rows * sizeof(int*)); // 행에 대한 포인터 배열 할당
for (int i = 0; i < rows; i++) {
    arr[i] = malloc(cols * sizeof(int)); // 각 행마다 열에 해당하는 메모리 할당
}
```
