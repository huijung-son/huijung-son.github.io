### 1. 변수

- 전역 변수는 초기화 되지만 지역 변수는 왜 초기화 되지 않는가
  - 효율성 때문이라 한다, 저장되는 메모리 위치도 다른 거 같고
  - 변수 초기화를 신경쓰자

### 3. 포맷

- printf와 같은 함수는 인자의 개수와 타입을 고정된 방식으로 정의하지 않고,<br>
첫 번째 인자인 포맷 문자열에 따라 동적으로 인자를 처리합니다.<br>
C 컴파일러는 이 가변 인자 목록에 대해 타입 검사를 수행하지 않으므로, 잘못된 포맷 형식을 지정해도 컴파일 에러가 발생하지 않는다.<br>
[GCC](https://gcc.gnu.org/onlinedocs/gcc/Warning-Options.html#index-Wformat)

| 형식   | 설명                           |
|------|------------------------------|
| %d   | 부호 있는 10진 정수 출력              |
| %c   | 단일 문자 출력                     |
| %p   | 포인터(메모리 주소) 출력               |
| %x   | 소문자 형태의 16진수 정수 출력           |
| %X   | 대문자 형태의 16진수 정수 출력           |
| %o   | 8진수 정수 출력                    |
| %s   | 문자열 출력                       |
| %u   | 부호 없는 10진 정수 출력              |
| %ld  | long 타입 부호 있는 10진 정수 출력      |
| %lld | long long 타입 부호 있는 10진 정수 출력 |

### 4. 음수, 보수법

- 컴퓨터는 음수를 표현하기 위해 메모리에 2의 보수법을 적용한 비트를 저장한다

### 5. 리터럴, 상수

- 리터럴: 변수에 할당될 수 있는 고정값
- 상수: 값을 변경 할 수 없는 변수

### 6. 부동소수점 오차

- 부동소수점 수는 가수 부분이 유한한 비트 수로 표현되기 때문에, 실제 무한소수 값을 근사치로 저장하게 된다
- 부동소수점 오차 줄이는 방법
  - 정수, 고정 소수점 사용
  - 안전한 알고리즘 또는 라이브러리 사용
